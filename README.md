# Animated Login Form using HTML and CSS

This repository contains an animated login form built with only HTML and CSS, featuring an interactive background and smooth animations. It’s perfect for adding a visually appealing login component to your website.

## Demo

You can view a live demo of the form [here](#).

## Features

- **Animated Background**: Floating elements create a dynamic background.
- **Interactive Form**: Input fields change styles based on focus and content.
- **Responsive Design**: Form layout adjusts for different screen sizes.
- **Only HTML & CSS**: No JavaScript required.

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/animated-login-form.git
   ```

2. Navigate into the project directory:

   ```bash
   cd animated-login-form
   ```

3. Open `index.html` in your browser to view the login form.

## Code Structure

### `index.html`

The main HTML file contains the structure of the login form, background elements, and spans for animation.

### `style.css`

The CSS file includes styles for:
- **Container and Background**: Positions and animates background elements.
- **Login Form**: Styles form inputs, buttons, labels, and interactive states.
- **Animations**: Defines `@keyframes` for background element animation.

### Fonts

The project uses the **Poppins** font from Google Fonts, which is imported directly in the CSS.

## Usage

You can use this login form by copying the HTML and CSS files into your project. Customize colors, animations, and other design aspects by modifying the `style.css` file.
